#### 2.1.2 字符串的“+”操作（字符串相加）

当“ + ”操作当中出现字符串，这个“ + ”就变成了**字符串连接符**，把前后进行拼接，形成一个新的字符串。

连“ + ”操作，从左到右。

```java
int age = 18;
System.out.println("我的年龄是" + "age" + "岁");// 我的年龄age岁
System.out.println("我的年龄是" + age + "岁");// 我的年龄是18岁
```

#### 2.1.3 字符的“+”操作（字符相加）

### 3.2 分支结构

#### 3.2.1 if判断

c语言

if (关系表达式) {
    语句体1;
} else if {
    语句体2;
} else if {
    语句体3;
} ...
else {
    语句体;
}

如果语句体只有一句，可以不加{}。
最好加，因为一句不好判断。
`int a = 10;`其实是两句，先定义，后赋值。

布尔类型变量boolean，可以直接把变量写在括号()里面。

#### 3.2.2 switch选择

c语言

switch(表达式) {
    case 值1:
    语句体1;
    break;
    ...
    default:
    语句体;
    break;
}

case 后面的**值**只能是字面量，且不能相同。

##### 3.2.2.扩展

1. default位置与省略。省略则无语句，位置不改变顺序。
1. case穿透。没有break,会穿透，直到遇到break。
1. (JDK 12)switch新特性。

    ```java
    switch (表达式) {
        case 值1 -> {
            语句体1;
        }
        ...
        default -> {
            语句体;
        }
    }
    ```
    如果语句体只有一句，可以不加{}。

(JDK 14)可以用变量去接收switch的结果。
yield 返回结果。

## Java内存分配

- 栈：方法运行时使用的内存
- 堆：存储对象或数组，new建立的
- 方法区：存储可以运行的class文件
- 本地方法栈：JVM在使用操作系统功能时使用
- 寄存器：给CPU使用

（堆与方法区连一起）

**JDK8开始，取消方法区，新增元空间，把全赖的方法区的多种功能进行拆分，有的功能在堆区，有的功能在元空间。**
